Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout "branch-name" to switch the active branch. (git checkout main)
  • git commit -m: sets the commit's message.
  • git-restore --staged "file-name": restores files in the working tree from either index or another commit.
  • git add: adds a change in the working directory to the staging area.
  • git push is utilized to send the committed changes to remote repositories for collaboration.
  • git branch: shows list of branches
  • git add * tracks all files.
  • git stash removes all of your local changes to those files.
  • git pull gets the files from the server.
