HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • !DOCTYPE HTML to begin a new HTML document
  • Unordered lists(ul) and ordered lists(ol) are very useful! Just make sure to include list items(li) inside their brackets!

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are 100's of colors with (:color), however when you start using hashtags(#9b9b9b) that number jumps to many more combinations!

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout "branch-name" to switch the active branch. (git checkout main)
  • git commit -m: sets the commit's message.
  • git-restore --staged "file-name": restores files in the working tree from either index or another commit.
  • git add: adds a change in the working directory to the staging area.
  • git push is utilized to send the committed changes to remote repositories for collaboration.
  • git branch: shows list of branches
  • git add * tracks all files.
  • git stash removes all of your local changes to those files.
  • git pull gets the files from the server.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
